# Allowed file types

## OVEWVIEW

Due to security issues, Minehut does not allow you to upload certian file types and in certian directories. For example you are unable to upload your own plugins to your Minehut server.

## ALLOWED FILE TYPES

```text
.yml
.json
.txt
.sk
.nbt
.mcfunction
.schematic
.properties
.csv
.png
.midi
.nbs
.mcmeta
.dat
.conf

```
